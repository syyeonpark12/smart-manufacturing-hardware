# Arduino 기반 전자부품 자동 검사·분류 시스템

컨베이어를 따라 이동하는 전자부품을 센서로 감지하고, PC에서 전달받은 검사 결과에 따라 서보모터로 자동 분류하는 프로젝트입니다.

저는 전체 하드웨어의 구조 설계, 부품 선정, 배치, 배선, 제어 코드 작성과 장비 테스트를 담당했습니다.

---

## 프로젝트 소개

전자부품이 컨베이어를 따라 이동하면 적외선 센서가 부품의 위치를 감지합니다.

부품이 검사 위치에 도착하면 컨베이어가 잠시 정지하고, PC의 검사 프로그램이 전자부품의 상태를 확인합니다.

검사 결과가 Arduino로 전달되면 결과에 맞는 서보모터가 작동하여 전자부품을 각각의 분류 위치로 이동시킵니다.

분류가 끝나면 서보모터가 원래 위치로 돌아가고 컨베이어가 다시 작동하여 다음 전자부품을 검사합니다.

## 동작 순서

1. PC에서 시작 명령을 보내면 컨베이어가 작동합니다.
2. 전자부품이 컨베이어를 따라 검사 위치로 이동합니다.
3. 적외선 센서가 전자부품을 감지합니다.
4. 전자부품이 검사 위치에 도착하면 컨베이어가 잠시 정지합니다.
5. PC의 검사 프로그램이 전자부품의 정상 여부를 확인합니다.
6. PC가 검사 결과를 Arduino로 전송합니다.
7. Arduino가 검사 결과에 맞는 서보모터를 작동시킵니다.
8. 서보모터가 불량인 전자부품을 해당 분류 위치로 쳐냅니다.
9. 작동이 끝나면 서보모터가 원래 위치로 돌아갑니다.
10. 컨베이어가 다시 작동하여 다음 전자부품을 검사합니다.
11. 같은 과정을 반복하여 여러 전자부품을 연속으로 검사하고 분류합니다.
12. 비상정지 버튼이 눌리면 컨베이어와 서보모터를 정지시키고 부저를 울립니다.

## 파일 구성

| 파일 | 내용 |
|---|---|
| `manufacturing_hardware.ino` | 핀 설정, 공통 변수, `setup()`과 `loop()` |
| `Communication.ino` | PC 시리얼 통신과 명령 처리 |
| `ConveyorControl.ino` | 적외선 센서 감지와 컨베이어 모터 제어 |
| `ServoControl.ino` | 서보모터 제어 동작 |
| `EmergencyStop.ino` | 비상정지 버튼과 부저 |
| `2차 프로젝트 회로도.png` | 하드웨어 연결 회로도 |

## 하드웨어 구성

| 부품 | 역할 |
|---|---|
| Arduino MEGA 2560 | 센서 입력과 전체 장비 제어 |
| DC Motor | 컨베이어 벨트 구동 |
| Motor Driver | DC 모터의 속도와 방향 제어 |
| IR Sensor | 전자부품 위치와 도착 여부 감지 |
| Servo Motor | 검사 결과에 따른 전자부품 분류 |
| Emergency Stop Button | 비상 상황에서 장비 동작 정지 |
| Buzzer | 비상정지 및 이상 상태 알림 |
| DHT22 | 장비 주변 온도 측정 |
| Battery Pack | 모터와 구동부 전원 공급 |
| Breadboard | 센서 및 전원 배선 연결 |
