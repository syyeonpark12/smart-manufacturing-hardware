## 프로젝트 소개

PCB 투입, 검사, 불량품 배출, 정상품 적재 및 창고 운반 과정을 자동화한 프로젝트입니다.

첫 번째 로봇팔이 PCB 기판을 메인 컨베이어에 올리면 AI 비전 시스템이 기판의 정상 여부를 검사합니다. 불량 PCB는 실린더를 이용해 컨베이어 옆으로 배출하고, 정상 PCB는 컨베이어 끝까지 이동시킵니다.

컨베이어 끝에 있는 두 번째 로봇팔은 정상 PCB를 집어 팔레트 이송형 컨베이어 위의 박스에 적재합니다. 박스에 정해진 수량의 PCB가 쌓이면 팔레트 이송형 컨베이어가 박스를 AGV 지게차 위치까지 이동시키고, AGV 지게차가 박스를 창고로 운반합니다.

운반을 완료한 AGV 지게차는 다시 초기 위치로 복귀하며, 새로운 빈 박스가 공급되면 전체 공정을 반복합니다.

## 동작 순서

1. 투입 위치의 로봇팔이 PCB 기판을 집어 메인 컨베이어 벨트 위에 올립니다.
2. 메인 컨베이어가 PCB 기판을 검사 위치까지 이동시킵니다.
3. 카메라와 AI 비전 시스템이 PCB의 정상 여부를 검사합니다.
4. 검사 결과가 불량이면 실린더가 작동하여 불량 PCB를 컨베이어 옆으로 밀어냅니다.
5. 검사 결과가 정상이면 정상 PCB는 메인 컨베이어 끝까지 계속 이동합니다.
6. 메인 컨베이어 끝에 있는 로봇팔이 정상 PCB를 집습니다.
7. 로봇팔이 정상 PCB를 팔레트 이송형 컨베이어 위에 놓인 박스에 적재합니다.
8. 2개의 PCB가 적재되면 팔레트 이송형 컨베이어가 작동합니다.
9. 팔레트 이송형 컨베이어가 PCB가 담긴 박스를 AGV 지게차의 인수 위치까지 이동시킵니다.
10. AGV 지게차가 박스를 들어 올린 뒤 라인을 따라 창고로 이동합니다.
11. AGV 지게차가 창고의 지정된 위치에 박스를 내려놓습니다.
12. 운반을 완료한 AGV 지게차는 다시 컨베이어 앞의 초기 위치로 복귀합니다.
13. 같은 공정을 반복합니다.
14. 비상정지 신호가 발생하면 컨베이어, 로봇팔, 실린더 및 AGV의 동작을 정지합니다.


## 파일 구성

| 파일 | 내용 |
|---|---|
| `manufacturing_hardware.ino` | 핀 설정, 공통 변수, setup과 loop |
| `Communication.ino` | PC 시리얼 통신과 명령 처리 |
| `ConveyorControl.ino` | 센서 감지와 컨베이어 모터 제어 |
| `ServoControl.ino` | 서보모터 분류 동작 |
| `EmergencyStop.ino` | 비상정지 버튼과 부저 제어 |
| `2차 프로젝트 회로도.png` | 전체 하드웨어 연결도 |

## 하드웨어 구성

| 부품 | 역할 |
|---|---|
| Arduino MEGA 2560 | 전체 하드웨어 제어 |
| DC Motor | 컨베이어 구동 |
| Motor Driver | 모터 속도와 방향 제어 |
| IR Sensor | 전자부품 감지 |
| Servo Motor | 검사 결과에 따른 분류 |
| Emergency Stop Button | 비상 상황에서 정지 |
| Buzzer | 비상 상태 알림 |
| DHT22 | 온도 측정 |
| Battery Pack | 구동부 전원 공급 |
