# Arduino 기반 전자부품 자동 검사·분류 시스템

컨베이어를 따라 이동하는 전자부품을 센서로 감지하고, PC의 검사 결과에 따라 서보모터로 분류하는 하드웨어 제어 프로젝트입니다.

Arduino MEGA 2560을 이용해 컨베이어 모터, 적외선 센서, 서보모터, 비상정지 버튼과 부저를 제어했습니다.

## 동작 순서

1. PC에서 시작 명령을 보내면 컨베이어가 움직입니다.
2. 적외선 센서가 전자부품을 감지합니다.
3. 전자부품이 검사 위치에 도착하면 컨베이어가 잠시 정지합니다.
4. PC에서 검사 결과를 Arduino로 보냅니다.
5. 결과에 맞는 서보모터가 동작해 부품을 분류합니다.
6. 분류가 끝나면 다음 부품 검사를 진행합니다.
7. 비상정지 버튼이 눌리면 모터를 멈추고 부저를 울립니다.

## 주요 기능

- 적외선 센서를 이용한 부품 감지
- 컨베이어 모터 시작 및 정지
- 검사 결과에 따른 서보모터 분류
- 비상정지 버튼과 부저 경고
- PC와 Arduino 사이의 시리얼 통신
- 온도값과 시스템 상태 전송
- PC 통신이 끊기면 컨베이어 정지

## 파일 구성

| 파일 | 내용 |
|---|---|
| `manufacturing_hardware.ino` | 핀 설정, 공통 변수, setup과 loop |
| `Communication.ino` | PC 시리얼 통신과 명령 처리 |
| `ConveyorControl.ino` | 센서 감지와 컨베이어 모터 제어 |
| `ServoControl.ino` | 서보모터 분류 동작 |
| `EmergencyStop.ino` | 비상정지 버튼과 부저 제어 |
| `2차 프로젝트 회로도.png` | 전체 하드웨어 연결도 |

## 하드웨어 구성

| 부품 | 역할 |
|---|---|
| Arduino MEGA 2560 | 전체 하드웨어 제어 |
| DC Motor | 컨베이어 구동 |
| Motor Driver | 모터 속도와 방향 제어 |
| IR Sensor | 전자부품 감지 |
| Servo Motor | 검사 결과에 따른 분류 |
| Emergency Stop Button | 비상 상황에서 정지 |
| Buzzer | 비상 상태 알림 |
| DHT22 | 온도 측정 |
| Battery Pack | 구동부 전원 공급 |
